Scripts and files to build NetBSD "Teokure Live Image",
pre-installed NetBSD Live Image for x86 PC with USB sticks
(or x86 virtual machines) with typical Japanese environments
and useful standard pkgsrc applications, including browser (firefox)
and Twitter client (mikutter) etc.

See http://www.ceres.dti.ne.jp/tsutsui/netbsd/liveimage/ for details.
(the page is in Japanese)
